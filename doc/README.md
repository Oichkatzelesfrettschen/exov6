# Reference Materials

This directory contains reference materials including specifications, standards, and books that are used for development but are not part of the project documentation.

## Contents

- **ben-books/** - POSIX and C standard reference books and specifications (PDFs)
  - POSIX specifications
  - C standard drafts
  - SUSv4/SUSv5 standards

- **standards/** - Official standard documents
  - C17 standard draft
  - SUSv5 HTML documentation
  - POSIX-2018 specifications

## Note

These are large binary files (PDFs, archives) totaling approximately 98MB. They are kept here for reference during development but are not required to build or run the kernel.

For project documentation, see the `/docs` directory at the repository root.
